# looplet Coder

Use this bundle when the user wants an observable coding agent that can
inspect a workspace, edit files, run tests, and iterate until the task is
complete.

The bundle preserves the behavior of the existing `examples/coder/agent.py`
example by compiling into normal looplet tools, hooks, memory sources,
compaction, and state.
